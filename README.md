# antolindavila.com (static site)

Sitio renovado del escritor Antolín Dávila, listo para publicarse en GitHub Pages.

## Estructura

- `index.html`: nueva web moderna
- `styles.css`: estilos de la nueva web
- `legacy/`: copia íntegra del sitio histórico original
- `.nojekyll`: desactiva Jekyll para servir todos los archivos legacy sin exclusiones

## Publicar en GitHub Pages

1. Sube este repositorio a GitHub.
2. Ve a **Settings > Pages**.
3. En **Build and deployment**, elige **Deploy from a branch**.
4. Selecciona la rama (normalmente `main`) y carpeta `/ (root)`.
5. Guarda y espera el despliegue.
